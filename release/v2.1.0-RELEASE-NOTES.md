# BiByte 2.1.0

## Highlights
- README completo (sumário navegável, tabelas, cookbook)
- Empacotamento: jpackage (EXE/MSI/DMG/DEB) + Launch4j (EXE slim)
- jlink: JRE portátil mínima para distribuir sem exigir Java instalado
- Gradle tasks de um comando para empacotar e preparar release local

## Artefatos
- JAR: BiByte-2.1.0.jar
- Latest alias: BiByte-latest.jar
- Windows: out/win/* (EXE/MSI e EXE slim + jre/)
- macOS: out/mac/*.dmg
- Debian: out/deb/*.deb

## Como usar
- Java: `java -jar BiByte-2.1.0.jar`
- Windows EXE slim: executar `BiByte-2.1.0.exe` (requer `jre/` ao lado)
- Windows MSI/EXE auto-contidos: instalar/rodar
